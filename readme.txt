Optimize your WordPress database tables with one click.

== Description ==

Optimize your database tables without phpMyAdmin in one click for high performance.

*Feature List*

* Easy Install and Manage.
* Automatically optimizes your database tables
* Only 4 kb

== Installation ==

1. Upload 'wp-db-optimizer' folder to the `/wp-content/plugins/` directory
2. Activate the plugin within you WordPress Administration Backend
3. Go to Manage > Optimize Database

If you have to upgrade manually simply repeat the installation steps.
